# Single-cell RNAseq Analysis in Google Cloud Platform using R

"Single cell RNAseq Data processing using R and RStudio in Google Cloud Platform"

# Quick start
1. Create VM instance in Google Cloud platform  as explained in "gcp_vm_setup.Rmd"

2. Create Cloud storage bucket and Mount to VIM instance for data storage and processing directly

3. Add the official CRAN repo and Seurat and Seurat dependencess as in "install_R_Seurat_GCP.Rmd"

4. Run Rscript to process single cell RNAseq from command line or use R to oprn R GUI
